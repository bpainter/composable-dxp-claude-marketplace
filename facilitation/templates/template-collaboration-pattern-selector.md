# Collaboration pattern selector

Quick chooser across the Collaboration Code patterns (100s–330s) for the design problem in front of you. Use during agenda design with the Sponsor Design Team.

For the full pattern catalog see [`book-collaboration-code-patterns.md`](../references/book-collaboration-code-patterns.md).

## What's the design problem you're trying to solve?

### "I don't know if the room is ready / set up right"

Reach for the **100s — Physical Environment** and **120s — Energetic Environment**.

| If… | Pattern |
|---|---|
| The room feels generic and corporate | 101 A place to be fully alive |
| Everyone won't fit comfortably | 102 Room for the whole system |
| Walls and tables are dead surfaces | 104 Everything speaks |
| The room needs to feel different from yesterday | 105 Fresh reusability |
| The energy in the room feels low or fragile | 122 Energetic container |
| You want people to take real risks | 132 Brave space |
| Music is missing or feels inappropriate | 130 More than a soundtrack |

### "I'm not sure who should be in the room"

Reach for the **200s — Preparation** and **210s — Participants**.

| If… | Pattern |
|---|---|
| The session feels under-mobilized | 203 Don't meet — MOBILIZE! |
| The sponsor team isn't a co-designer | 205 Sponsor Design Team as microcosm |
| Critical perspectives are missing | 211 Get the whole system in the room |
| One expert is going to dominate | 213 Careful use of experts |
| Breakout sizes feel wrong | 215 The right sizes for breakout groups |
| Diversity isn't designed into the room | 216 Diversity and inclusion |
| Some participants are remote | 217 Virtual participation |

### "I'm worried about the agenda design itself"

Reach for the **220s — Workshop Design**.

| If… | Pattern |
|---|---|
| The design feels too tame for the problem | 221 Requisite variety of design |
| The shape doesn't repeat at every scale | 222 Recursion |
| You're skipping prototypes for debate | 223 Iteration and rapid prototyping |
| The agenda feels too safe | 224 It's not safe to play it safe |
| You're cramming too much in | 225 Go slow to go fast |
| You haven't planned the body's needs | 228 Rhythms of the body and the day |
| The session is overnight | 229 Sleep on it (twice!) |
| You're afraid an activity will fail | 234 Graceful failure |
| Virtual or hybrid format | 235 Designing Virtual Sessions |

### "I'm worried about how the room will move"

Reach for the **240s — Process**.

| If… | Pattern |
|---|---|
| You're learning as you go | 241 Lead (and learn) with experience |
| You want self-directing teams | 242 Sapiential leadership (self-facilitating teams) |
| The room is stuck | 243 Move the group |
| Debate is going in circles | 244 Don't argue. Test! |
| The session needs play | 245 Work as play |
| You need a mid-session redesign | 250 Real-time re-design |
| Resistance is rising | 253 Honor resistance |

### "I'm worried about closure and what gets carried forward"

Reach for the **260s — Outcomes and Deliverables**.

| If… | Pattern |
|---|---|
| The work needs to outlive the event | 261 Make the solution last past Act Day |
| You need a felt sense of ending | 263 Rituals of closure |
| The deliverable feels like a slide deck | 264 Engaging deliverables |

### "I'm worried about myself as the facilitator"

Reach for the **300s — Learning Facilitation** and **320s — Mastering Facilitation**.

| If… | Pattern |
|---|---|
| Pre-event nerves | 301 Preparation is the antidote to fear |
| You're making it about you | 303 It's not about YOU |
| You want to abandon the design mid-session | 305 Trust in the process (and the KreW) |
| You're avoiding the hardest part | 306 Move toward the fear |
| Difficult truth needs to be said | 308 Tell the truth with courage and compassion |
| You're co-facilitating | 309 Conscious co-facilitation |
| Energy in the room is wrong | 310 Shifting group energy |
| First time running this design | 311 Don't open on Broadway |
| Online format | 312 Facilitating Online DesignSessions |

## How to use this selector

1. With the Sponsor Design Team, walk through the six "What's the design problem" sections.
2. For each section, name the patterns that *aren't* honored in the current draft.
3. For each gap, decide: redesign, accept the risk, or escalate.
4. Note the patterns honored on the [session design template](template-session-design.md).
