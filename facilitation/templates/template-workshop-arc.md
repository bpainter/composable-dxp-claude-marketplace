# Workshop arc — choose your scale

Reference arcs at five common durations. Each one is Scan → Focus → Act recursively, sized to the time available. Use as the starting skeleton for any [`template-session-design.md`](template-session-design.md).

## 90-minute meeting

A focused decision in one sitting. The smallest unit where Scan/Focus/Act is meaningful.

| Time | Phase | Block |
|---|---|---|
| 00:00 | Open | Frame, purpose, decision rule (5 min) |
| 00:05 | Scan | Silent generation; share-out (20 min) |
| 00:25 | Scan→Focus | Theme the ideas; identify 3–5 worth deciding on (10 min) |
| 00:35 | Focus | Trade-off discussion (30 min) |
| 01:05 | Act | Vote / decide; assign owner; checkpoint date (15 min) |
| 01:20 | Close | Recap; commitment (10 min) |

## Half-day workshop (3.5 hours)

Common for kickoffs, charter alignment, and mid-engagement redirects.

| Time | Phase | Block |
|---|---|---|
| 00:00 | Open | Frame, norms, agenda, introductions (15 min) |
| 00:15 | Scan | Current-state mapping or empathy work (40 min) |
| 00:55 | Break | (10 min) |
| 01:05 | Scan→Focus | Theme; pick the 2–3 problems worth designing (30 min) |
| 01:35 | Focus | Solution generation; first-draft proposals (45 min) |
| 02:20 | Break | (10 min) |
| 02:30 | Focus | Trade-off analysis; convergence (30 min) |
| 03:00 | Act | Decisions; owners; communication plan (20 min) |
| 03:20 | Close | Recap; commitment ritual (10 min) |

## One-day workshop (8 hours including lunch)

A full Scan/Focus/Act in a single day. Common for vision sessions, sprint planning, single-decision strategy work.

| Time | Phase | Block |
|---|---|---|
| 09:00 | Open | Frame, sponsor opening, agenda, introductions (45 min) |
| 09:45 | Scan | Current-state, context, perspectives (90 min) |
| 11:15 | Break | (15 min) |
| 11:30 | Scan→Focus | Synthesis; problem framing (60 min) |
| 12:30 | Lunch | (60 min — keep it on-site if possible) |
| 13:30 | Focus | Solution sketching; small-group prototypes (90 min) |
| 15:00 | Break | (15 min) |
| 15:15 | Focus | Gallery walk / share; trade-off discussion (60 min) |
| 16:15 | Act | Decisions; owners; transformation roadmap (30 min) |
| 16:45 | Close | Recap; commitment statement; closing ritual (15 min) |

## Two-day workshop

Day 1 = Scan + start of Focus. Day 2 = finish Focus + full Act. Sleep-on-it (Pattern 229) between days.

### Day 1 — Open the field
| Time | Phase | Block |
|---|---|---|
| AM | Open | Sponsor frame, deep introductions, context |
| AM | Scan | Empathy / research / current-state |
| Lunch | Break | Working lunch — informal connection |
| PM | Scan | Stakeholder perspectives, possibilities |
| PM | Scan→Focus | Theme; problem framing |
| Close | — | Day 1 reflection; homework for Act Day |

### Day 2 — Narrow and commit
| Time | Phase | Block |
|---|---|---|
| AM | SDT | Sponsor Design Team morning circle-up (separate from main room) |
| AM | Focus | Design challenges; first-draft solutions |
| Lunch | Break | |
| PM | Focus | Second-draft solutions; trade-offs; convergence |
| PM | Act | Decisions; owners; communication plan; transformation roadmap |
| Close | — | Final report-out; commitment; closing ritual |

## Three-day workshop (the wavespace standard)

The most common Collaboration Code format. Day 1 = Scan. Day 2 = Focus. Day 3 = Act. SDT meets every morning and evening.

### Day 1 — Scan
| | Block |
|---|---|
| AM | Open, sponsor frame, full introductions, common language |
| AM | Empathy / research / current-state mapping |
| Lunch | Working / connection |
| PM | Stakeholder perspectives; possibilities; field trips if relevant |
| PM | Theme work; emerging problems |
| Evening | SDT circle-up; refine Day 2 |

### Day 2 — Focus
| | Block |
|---|---|
| AM | SDT morning circle-up |
| AM | Problem framing; design challenges |
| AM | Small-group solution sketching (multiple parallel teams) |
| Lunch | |
| PM | Trade-off analysis; convergence |
| PM | Second-draft solution; refinement |
| Evening | SDT Day Two evening circle-up; design Act Day; sleep on it |

### Day 3 — Act
| | Block |
|---|---|
| AM | SDT Act Day morning circle-up |
| AM | Final solution synthesis |
| AM | Buckets of work; team formation |
| Lunch | |
| PM | Transformation roadmap; communication plan; symbolic action |
| PM | Final report-out |
| Close | Commitment statement; closing ritual |

## Sizing rule of thumb

Length matches stakes and complexity, not preference:

| Scale | Stakes | Complexity | Examples |
|---|---|---|---|
| 90 min | Single decision | Low | Scope clarification, single design call |
| Half-day | One major output | Moderate | Project kickoff, single canvas |
| One day | Vision or strategy commit | Moderate-high | Quarterly planning, single-stream strategy |
| Two days | Multi-output engagement | High | Product strategy, M&A integration, transformation kickoff |
| Three days | Cross-functional transformation | High | Wavespace events, innovation center launches, major change programs |

## Notes on energy management

Across all scales:

- Build in a break every 75–90 minutes minimum
- Keep blocks under 60 minutes; people fade after that
- Lunch should be at least 45 minutes; don't over-schedule lunch as "working"
- Day 2 of multi-day events has the lowest energy; design the lightest blocks here
- Last block of any day should be lower-cognitive-load than the morning
