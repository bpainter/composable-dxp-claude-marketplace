## Notes on the index

- Numbering follows the EY wavespace edition (2021) of the Second Edition
- A few numbers are missing in the source (e.g., 204, 209, 219, 246, 254–259, 265–299, 313–319, 329) — reserved for additions and not all numbers are populated
- Cross-references in the source use these codes; preserve them when citing
- This index is for navigation only; the patterns themselves are described in the source volume (personally licensed to Bermon as wavespace professional)
