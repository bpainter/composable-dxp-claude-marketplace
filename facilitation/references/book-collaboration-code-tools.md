# Book digest: *The Collaboration Code — Tools*

**Author**: Rob Evans
**What this book is**: The complete catalog of workshop modules and tools — the *exercises* you assemble into an agenda. Where *Patterns* explains why, *Tools* tells you what to actually run.

## The organizing model: Scan → Focus → Act

Every collaborative session moves through three macro-phases. Tools are organized by which phase they belong to.

- **Scan** — open the field. Surface current state, possibilities, unknowns, perspectives. Divergent. The room generates more than it converges.
- **Focus** — narrow to what matters. Pick the few problems or solutions worth designing. Decision-heavy. Trade-offs surface.
- **Act** — commit and mobilize. Plans, owners, deliverables, communication. Convergent and forward-looking.

A 90-minute meeting has these three phases compressed; a three-day session has them as full days; a multi-week engagement has them as work streams.

## The full catalog (by section)

### Pre-Session Tools (~12)
The setup work that decides whether the session lands. Examples: Graphical Strawdog, Knowledge Wall, Mind Mapping, Online File Structure, Participant Invitation, Participant Pre-work, Scoping and Scaling Workshops, **Sponsor Design Team Planning Session**, Sponsor Design Team Prep Day Circle Up, Standard Assignment Page Format, Storyboard, Workshop Design Document.

### Reporting and Report Formats (~12)

### Deliverables (~5)

### Additional Reference (~6)

## How each tool is documented

Every tool entry follows the same structure:

- **What It Is** — brief description
- **What to Use It For** — when to reach for it
- **How It Works** — step-by-step delivery instructions
- **Example** — a real assignment from an actual session
- **Timing** — approximate duration (always "approximate" in handouts)
- **Keep in Mind** — design and delivery tips

This format is the gold standard for documenting workshop modules. The `template-session-design.md` template in this plugin uses a compatible structure.

## How to reach for this in real work

Three reads:

1. **Scoping a session** — skim Pre-Session Tools to make sure your prep is honest.
2. **Designing the agenda** — walk the Scan/Focus/Act sections in order. Pick three to five tools per phase based on the room's needs and the time available.
3. **Stuck mid-session** — the catalog has an answer for almost any room problem. Idea Rave for low energy. Fishbowl when one expert is dominating. Plus/Delta for end-of-day feedback.
