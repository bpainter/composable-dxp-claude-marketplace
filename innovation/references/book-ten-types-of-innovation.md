# Ten Types of Innovation — Keeley, Pikkel, Quinn, Walters / Doblin (2013)

**Source digest** for the innovation plugin. Cite as "(Keeley et al., *Ten Types*, p. X)" inside skill bodies.

## Thesis

Innovation isn't only about products. Across ~2,000 innovations analyzed over 100+ years, Doblin identified ten distinct *types* of innovation in three categories. The book's central empirical finding: **innovations that combine three or more types persist; single-type innovations get copied.** Most companies over-invest in Product Performance (Type 5) and ignore the seven types that produce sustained advantage.

## The Ten Types

### Configuration (internal — competitor-invisible, hardest to copy)

1. **Profit Model** — *how you make money.* Subscription, freemium, ads, marketplace fees, premium tiers, outcome-based pricing, leasing, financing, microtransactions.
2. **Network** — *who you partner with and how value flows.* Consortia, ecosystems, white-label, alliance models, supplier integration, franchising.
3. **Structure** — *how you organize talent and assets.* Corporate university, employee ownership, distributed teams, asset-light, talent network, IP licensing.
4. **Process** — *how you do work.* Lean, predictive analytics, on-demand production, just-in-time, IP strategies, process automation.

### Offering (the product itself)

5. **Product Performance** — *features, function, quality.* The most-overinvested type. Features, reliability, performance, ease of use, customization, sustainability.
6. **Product System** — *how products combine into a system.* Modularity, platforms, integrated suites, complementary products, extensions.

### Experience (customer-facing — visible, fastest to copy)

7. **Service** — *augment the offering with services.* Concierge, guarantees, total experience, lease/loaner, supplementary support.
8. **Channel** — *how you reach customers.* Omnichannel, direct, indirect, multi-modal, on-demand delivery, contextual presence.
9. **Brand** — *identity and meaning.* Component branding, certification, transparency, brand extension, private label, value alignment.
10. **Customer Engagement** — *interactions that build affinity.* Community, mastery enabling, autonomy/authority, identification, experience curation.

## Why combinations win

Single-type innovations produce short-term advantage that competitors copy quickly. Combinations across multiple types create *system effects*:

- **Cost** of copying scales non-linearly — copying one type is cheap; copying four is a full strategic re-architecture
- **Customer perception** of differentiation is higher with cross-type innovation
- **Internal coherence** strengthens — the combination implies a strategy, not a feature

Examples (canonical):

- **Apple iPod era** — Type 5 (device) + Type 6 (iTunes/iPhone integration) + Type 7 (Genius Bar) + Type 9 (brand). Decade-long lead.
- **Costco** — Type 1 (membership profit model) + Type 4 (limited SKU process) + Type 8 (warehouse channel) + Type 2 (member-supplier network). Configuration-deep, very hard to copy.
- **Cirque du Soleil** — Type 5 (no animals, theatrical product) + Type 8 (residency channel) + Type 9 (brand) + Type 10 (community). Created a category by combination.
- **Method** — Type 9 (brand) + Type 8 (Target retail) + Type 5 (formulation) + Type 10 (community). Won a category dominated by P&G/Unilever scale.

## The diagnostic — most companies are Product-Only

The book's critical applied insight: most enterprises score 0–1 high types and 8–9 invisible types. They're competing on Product Performance alone, which is structurally the most copyable type. Doblin's data suggests Configuration types (1–4) deliver disproportionate sustained advantage but receive disproportionately less innovation investment.

Common patterns surfaced by the diagnostic:

- **Product-only** — high Type 5, near-zero on 1, 7, 10. Vulnerable.
- **Channel-led** — high Type 8, low Type 5. Distribution that erodes.
- **Brand without backing** — high Type 9, low Configuration. Reputation outpaces operating model.
- **Configuration-deep** — high Types 1–4 (Costco, Southwest, Walmart). Most defensible.

## Innovation as discipline

The book's secondary thesis: innovation is a discipline, not a stroke of insight. Doblin's prescription:

- **Diagnose before generating** — score the org's current ten-type investment before brainstorming
- **Design for combinations** — explicitly seek 3–5 type combinations, not single-type bets
- **Refuse type imbalance** — if every brainstorm idea is Type 5, force the team into Configuration thinking
- **Use the typology as M&A / partnership lens** — surface what type the org needs to acquire vs. build

## Three patterns from the book worth quoting

- **"More than one is more than the sum"** — the empirical observation that combinations create disproportionate advantage
- **"Not all types are equal"** — Configuration types are usually most defensible; Experience types most customer-perceptible
- **"Innovation is what users see"** — internally novel work that doesn't change customer experience isn't a market innovation, even if it's an organizational one
