Use the `leadership-people-leader` and `leadership-meetings-and-cadences` skills to prep or log a 1:1.

## When the user invokes this

Ask one question if not provided: **"Which direct report?"**

## Steps

1. **Locate or create the running log.** Look for `10_Practice/People/Reports/{Name}/1on1_Log.md`. If missing, create from `70_Templates/T_1on1_Log.md` and note that it's a new log.
2. **Read the most recent 2–3 entries.** Surface what's still open from prior conversations.
3. **Surface from the running log:**
   - Open commitments (mine to them, theirs to me)
   - Standing topics (career conversation due? coaching focus active?)
   - Patterns from recent entries (what keeps coming up; what they're avoiding)
4. **Read their `_Profile.md` if it exists.** Pull motivation profile, growth edges, current coaching arc.
5. **If the user is *prepping*:**
   - Show the open commitments and standing topics
   - Suggest 2–3 specific questions for this slot (pull from the topic bank in `manager-tools-one-on-one.md`; tailor to their current state)
   - Flag anything overdue (career conversation hasn't happened in a quarter; promised follow-up never landed)
6. **If the user is *logging*:**
   - Add a new dated entry to the top of the log
   - Capture: their agenda, my agenda, future / growth, commitments out
   - Update open commitments and standing topics sections
   - Save the file

## Format

Follow the Manager Tools O3 structure: their 10 / my 10 / future 10. Don't restructure the log; append entries at the top.

## Don't

- Paste sensitive content into chat unless the user asks for the in-conversation render
- Drift into general feedback advice — route to the `/feedback` command if a feedback conversation is what's needed
- Skip surfacing open commitments — that's the highest-value thing this command does

## Pairs with

- `/feedback` — when the 1:1 needs a specific feedback delivery
- `/coach` — when the 1:1 is part of a coaching arc
- `/perspective` — if the 1:1 was hard and the user needs to decompress after
