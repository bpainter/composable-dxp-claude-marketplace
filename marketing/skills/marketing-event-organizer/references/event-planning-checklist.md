# Comprehensive Event Planning Checklist

## Pre-Event (4-8 Months Out)

### Strategic Planning
- [ ] Define event goals (attendance, engagement, revenue, brand awareness)
- [ ] Identify target audience and attendee personas
- [ ] Determine event format (in-person, hybrid, virtual)
- [ ] Set event date, duration, and general timeline
- [ ] Establish budget range and approval process
- [ ] Define success metrics and KPIs
- [ ] Research competitive events and differentiate
- [ ] Build event steering committee or leadership team

### Venue & Logistics
- [ ] List venue requirements (capacity, space, amenities, accessibility)
- [ ] Scout and visit 3-5 venues
- [ ] Negotiate contracts and terms
- [ ] Confirm parking, public transit, ADA accessibility
- [ ] Reserve date and secure contract signature
- [ ] Arrange liability insurance
- [ ] Identify permits or approvals needed
- [ ] Plan room configurations and flow

### Sponsorship & Revenue
- [ ] Design sponsorship tier structure (Gold/Silver/Bronze, etc.)
- [ ] Calculate ROI for each tier
- [ ] Create sponsorship prospectus (benefits, visibility, pricing)
- [ ] Develop sponsor contract and terms
- [ ] Identify and outreach to potential sponsors
- [ ] Set sponsorship sales targets and deadlines
- [ ] Plan sponsor activation and engagement

## Preparation (2-4 Months Out)

### Vendor Management
- [ ] Request RFPs from caterers (get 3 quotes minimum)
- [ ] Request RFPs from A/V companies
- [ ] Request RFPs from signage/design vendors
- [ ] Negotiate contracts with selected vendors
- [ ] Confirm vendor contracts and deposit payments
- [ ] Establish communication and check-in schedule
- [ ] Create vendor contingency contacts

### Marketing & Promotion
- [ ] Develop event branding and visual identity
- [ ] Build event website and registration page
- [ ] Create email campaign sequences (10-15 emails)
- [ ] Develop social media content calendar
- [ ] Create promotional graphics and videos
- [ ] Set up Google Ads and retargeting campaigns
- [ ] Identify and engage media partners
- [ ] Launch early bird registration and pricing

### Programming & Speakers
- [ ] Define event themes and tracks
- [ ] Create CFP (Call for Proposals) with deadline
- [ ] Build speaker recruitment list (30-50 targets)
- [ ] Begin speaker outreach and negotiations
- [ ] Establish speaker honorarium and logistics
- [ ] Confirm keynote speakers and secure bios/photos
- [ ] Create speaker welcome packet
- [ ] Plan speaker preview and tech-check process

### Community & Partnerships
- [ ] Identify strategic partners (media, associations, sponsors)
- [ ] Reach out and establish partnership terms
- [ ] Create partner benefits and co-promotion plan
- [ ] Build ambassador program and recruit ambassadors
- [ ] Establish partnerships with local communities
- [ ] Plan pre-event networking or workshops

## Final Preparation (2-4 Weeks Out)

### Attendee Experience
- [ ] Launch final registration push and send reminders
- [ ] Create attendee app or agenda platform
- [ ] Develop attendee pre-event emails (agenda, tips, parking)
- [ ] Send speaker bios and session descriptions to attendees
- [ ] Create signage templates (wayfinding, session signs)
- [ ] Confirm dietary requirements and accessibility needs
- [ ] Create attendee welcome packet (physical or digital)
- [ ] Set up networking opportunities (breakfast, reception, dinner)

### Logistics Finalization
- [ ] Confirm final headcount with venue
- [ ] Finalize catering menu and quantities
- [ ] Confirm A/V setup and test critical equipment
- [ ] Create run-of-show (minute-by-minute schedule)
- [ ] Design session schedule to avoid conflicts
- [ ] Confirm all speaker presentations received
- [ ] Create speaker introduction scripts
- [ ] Brief speakers on timing and logistics

### Volunteer & Staff
- [ ] Recruit and onboard volunteers
- [ ] Create volunteer role descriptions and assignments
- [ ] Hold volunteer orientation meeting (in-person or virtual)
- [ ] Assign staff to key roles (registration, tech support, etc.)
- [ ] Create staff checklists and communication protocols
- [ ] Plan staff meeting schedule for event day
- [ ] Confirm all staff/volunteers understand roles

### Marketing Final Push
- [ ] Send final registration deadline email
- [ ] Promote on social media daily
- [ ] Reach out to media for coverage
- [ ] Send speaker and sponsor thank-you messages
- [ ] Create event day social media content queue
- [ ] Set up event hashtag and monitoring
- [ ] Prepare speaker and attendee social graphics

## Event Day (Day-Of)

### Morning Setup (4-6 Hours Before)
- [ ] Arrive early and do facility walk-through
- [ ] Set up registration table with name badges
- [ ] Install signage and wayfinding
- [ ] Test A/V equipment (projectors, mics, sound)
- [ ] Conduct sound check with speakers
- [ ] Set up catering stations
- [ ] Position staff and volunteers
- [ ] Create backup power supplies for critical equipment

### Pre-Event (1 Hour Before)
- [ ] Confirm all speakers are present and ready
- [ ] Perform final A/V check
- [ ] Test registration system and badge printing
- [ ] Brief staff and volunteers on assignments
- [ ] Confirm first session speaker and timing
- [ ] Open registration early
- [ ] Start welcoming attendees and social media sharing

### During Event
- [ ] Monitor session timing and keep speakers on schedule
- [ ] Manage technology and troubleshoot issues
- [ ] Capture photos and video content
- [ ] Monitor social media mentions and respond
- [ ] Track attendance numbers
- [ ] Manage speaker transitions
- [ ] Ensure volunteers are in position and engaged
- [ ] Collect attendee feedback throughout (mobile app, QR codes)

### Post-Event (Within 1 Hour)
- [ ] Thank speakers and sponsors publicly
- [ ] Collect final feedback surveys
- [ ] Document lessons learned with team
- [ ] Secure all equipment and materials
- [ ] Debrief with venue and vendors
- [ ] Collect final photographs
- [ ] Reconcile budget and expenses

## Post-Event (Within 2 Weeks)

### Attendee & Community
- [ ] Send thank you email to all attendees
- [ ] Share photos and video highlights
- [ ] Publish session recordings (if applicable)
- [ ] Send speaker and sponsor thank you gifts
- [ ] Request testimonials from speakers and sponsors
- [ ] Publish post-event blog post or recap
- [ ] Plan follow-up engagement (newsletter, next event)

### Analysis & Reporting
- [ ] Compile attendee feedback and NPS scores
- [ ] Calculate attendance and engagement metrics
- [ ] Analyze social media reach and engagement
- [ ] Track sponsor ROI and follow-up conversations
- [ ] Calculate event profitability and ROI
- [ ] Document lessons learned and improvements
- [ ] Create event case study and takeaways
- [ ] Share results with sponsors and stakeholders

### Planning for Next Event
- [ ] Schedule debrief meeting with team
- [ ] Document what worked and what to improve
- [ ] Gather budget-related information for next year
- [ ] Start planning next event (date, theme, goals)
- [ ] Build relationships from this event into next
- [ ] Archive all event materials for reference

## Budget Categories to Include

- Venue rental and setup fees
- Catering (food, beverages, service)
- Audio/visual and technology
- Signage, printing, branded materials
- Speaker travel and accommodations
- Staff and volunteer appreciation
- Marketing and promotion
- Insurance and permits
- Contingency (10-15% of total)
- Event platform or registration software
- Photographer/videographer
- Entertainment or special activities
- Contingency fund for unexpected costs

## Common Vendor Contacts to Establish

- Caterer (ask for menu options and pricing)
- AV/Technology provider
- Signage and design
- Printing and promotional items
- Photographer/videographer
- Florist or event designer (if applicable)
- Security and parking
- Transportation (shuttle, valet)
- Hotel group (for out-of-town attendees)

## Key Metrics to Track

- Registration numbers and conversion rate
- Email open and click-through rates
- Social media reach, impressions, engagement
- Attendance (actual vs. registered)
- Session attendance by track
- Speaker satisfaction and feedback
- Attendee satisfaction (NPS, survey results)
- Sponsor satisfaction and ROI
- Media coverage and impressions
- Budget variance from plan
- Volunteer engagement and satisfaction
