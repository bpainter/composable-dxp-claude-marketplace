# marketing

**Marketing and content: copywriting, search, growth, analytics, social, and events**

## Skills (8)

| Skill | What it does |
|---|---|
| [[marketing-copywriter]] | Skilled digital copywriter who writes clear, compelling, actionable copy for modern web experiences across B2B and B2C — landing pages, UX microcopy, conversion-focused content. |
| [[marketing-seo-geo-strategist]] | SEO/GEO/AEO strategist for modern web architectures. Traditional search optimization plus generative-engine and answer-engine optimization for ChatGPT, Claude, Perplexity, Google AI Overviews. |
| [[marketing-seo-brief]] | Produces SEO content briefs a writer can hand off and execute against — primary and secondary keywords, search intent, SERP feature targets, competitor coverage, outline with H-tag hierarchy, internal linking. |
| [[marketing-geo-content]] | Structures content so AI assistants and answer engines (ChatGPT, Claude, Perplexity, Google AI Overviews, Bing Copilot) can extract, attribute, and cite it cleanly. |
| [[marketing-growth-hacker]] | Marketing-driven growth strategist focused on accelerating user and revenue growth through creative, data-informed experimentation and channel optimization across the funnel. |
| [[marketing-analytics-strategist]] | Web analytics architect using GA4, Google Tag Manager, and custom event tracking. Implementation, measurement plans, and dashboards that connect to business outcomes. |
| [[marketing-social-media-marketer]] | Seasoned social media strategist who builds authentic, high-converting native content strategies for LinkedIn and Instagram. |
| [[marketing-event-organizer]] | Event organizer and community builder planning conferences, meetups, and gatherings with strategic clarity and operational precision. |

## Agent

**`marketing-agent`** — Senior marketer who knows when to use each skill in this plugin and how to sequence them. Use this agent when a task spans strategy → execution (e.g., positioning → SEO brief → copy → analytics), requires sequencing, or needs senior judgment about which skill applies.

## Conventions

- Each skill has a `SKILL.md` with YAML frontmatter and structured sections.
- Each skill has a `references/` folder with skill-specific reference files (where present).
- Tone across all skills: direct, practical, builder-oriented.

## Pairs with

- **`cx`** — `cx-personalization-strategist` and `cx-customer-research` inform targeting and messaging
- **`design`** — landing-page design and conversion-optimized layouts
